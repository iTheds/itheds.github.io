基本概念

质量检验的主要功能：把关、预防、报告、改进。 把关是质量检验最基本的职能，也可称为质量保证职能。 这一职能是质量检验出现时就已经存在的，即使是生产自动化高度发展的将来，检验的手段和技术有所发展和变化，质量检验的把关作用，仍然是不可缺少的。

质量检验机构的权限（八项权限）
1）有权在组织内认真贯彻产品质量方面的方针、政策，执行检验标准和有关技术标
2）有权按有关技术标准的规定，判定产品和零部件合格与否；
3）有权按有关规定，对购入的各种原材料、外购件、外协件及配套产品等进行检验 ，并确定其合格与否；
4）对交验的产品或零部件，当缺乏相应的标准或技术文件时，有权拒绝接收检验；
5）有权对产品或零部件的材料代用等，参与分析、研究和审批；
6）对于忽视产品质量，以次充好，弄虚作假等行为，有权制止，限期改正，建议视 其情节轻重给予责任者以相应的处分；
7）有权追查产品质量事故产生的原因，找出责任者，提出视情节给予处分的建议；
8）对产生的各类不合格品，有权如实进行统计与分析，要求责任部门针对存在的问 题提出相应的改进措施。

标准实样 standard model
对零、部（组）件或成品难以通过监视和测扯设备鉴别的质量特性，经选定和批准作为 检测判定依据 的样品

组织应依据产品实现策划的安排，组织实施产品检验工作，并保存 产品符合接受准则的证据。4.2 检验人员的配置应满足组织检验工作要求。

检验规范（大纲、细则）的编制：
需符合国家标准、军用标准的要求（格式、内容）
应包括合同（技术协议）要求的全部检验项目
应与设计文件、工艺文件保持一致
应对检验活动作出具体规定（人、机、料、法、环）

质量检验机构职责

对由于 未认真 贯彻执行产品质量方面的方针、政策、标准或规定，致使产品质量低劣或出现质量事故负一定责任
对不合格品管理不善所造成的后果负责
对统计上报的质量报表、质量信息的正确性、及时性负责对因 错检、漏检、误检 而造成的质量损失负责
对忽视质量的行为或质量事故，不反映、不上报，而造成的影响和后果负责
对明知是不合格品，还给予检验并签发检验合格证书的行为负责

依据质量要求， 独立 判定产品合格与不合格的权利
对于不具备检验条件或要求的产品，有权 拒绝 接收检验
对于忽视产品质量，以次充好，弄虚作假等行为，有权制止、限期改进及视情节提出处分的建议
对产品质量事故，有权 追查 产生的原因，找出责任者，视情况提出给予处分的建议
对生产中产生的各种不合格品，有权如实进行统计与分析，针对存在问题，要求有关责任部门提出 改进
对国家明令禁止的淘汰产品、以假充真的产品，有权不予检验
对研制生产过程控制进行监督检查

质量检验机构在独立行使质量检验职能时，不受内外部任何方面的干扰，客观、公正行使职权。

检验部门 的职责和权限至少应包括:

a) 负责检验工作的归口管理，编制或参与编制检验文件，会签有关技术文件。
b) 根据检验文件组织检验人员对产品进行检验，做出符合性的结论，办理质量证明文件。
c) 。 按规定保存产品检验的相关证明性文件。 一般包括 :
、 产品合格证明、 产品检验证据、不合格品处置记录等。
d) 负责或参与检验人员的培训和考核，确保检验人员按
要求持证上岗。
e) 负责检验印章的管理。
f) 按规定的授权处置不合格品，参与不合格品原因分析。
g) 按规定参与首件鉴定、技术攻关、产品质量评审等工
作。必要时，参与对供方的质量保证能力考察。
h) 有权越级反映质量问题。

检验人员 的职责和权限至少应包括:

a) 依据检验文件、设计文件、工艺文件以及合同( ( 或技术
协议) ) 等产品质量要求，对采购产品、过程产品和最终产品做出
合格否判断，并对检验结果的与正确性负责；
b) 负责对产品检验状态进行标识；
c) 做好检验、试验记录并及时归档；
d) 有权越级反映质量问题。
